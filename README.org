   #+END_SRC

2. **Clone the Configuration**:
   #+BEGIN_SRC bash
   git clone git@github.com:joppliger/dotemacs.git ~/.config/emacs
   #+END_SRC

3. **Start Emacs**: Run ~emacs~ and let it install the necessary packages and dependencies.
